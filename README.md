# 🗡️ RPG Adventure Game

A 2D pixel-art RPG adventure game developed with **Godot 4** and **GDScript**. The project is currently in active development and focuses on creating an engaging world with NPC interactions, enemy AI, and story-driven dialogue.

> 🚧 **Status:** In Development

---

## 📖 About

This project is a personal game development journey where I am building an RPG from scratch while learning game programming and software architecture.

The game features interactive NPCs, enemy encounters, and a dialogue system powered by JSON files for easy content management.

---

## ✨ Current Features

- 🎮 Player movement
- 💬 JSON-based dialogue system
- 👤 NPC interaction
- 👹 Enemy AI using Area2D detection
- ⚔️ Enemy attack behavior
- 🎵 Background music
- 🏞️ Multiple game scenes
- 🎨 Pixel art environment

---

## 🛠️ Built With

- **Engine:** Godot 4
- **Language:** GDScript
- **Dialogue:** JSON
- **Version Control:** Git & GitHub
- **Art Style:** 2D Pixel Art
